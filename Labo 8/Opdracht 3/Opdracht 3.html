Opdracht 3

Vraag: Waarom staat de tekst in de blokken horizontaal gecentreerd?

Antwoord: Dit komt omdat in de CSS van de blokken de CSS code 'text-align: center' staat.


Vraag: Waarom heeft de property 'vertical-align: center' geen effect?

Antwoord: Dit komt omdat het enkel invloed heeft op table-cell en inline elementen, het heeft geen effect op de inhoud ervan maar enkel op de inhoud + er rond.


Vraag: Ga na dat vertical-align:middle er niet voor zorgt dat de tekst verticaal gecentreerd word. Wat doet het dan wel?

Antwoord: 'Vertical align: middle' past niet de inhoud van het element aan, maar het element zelf.


Vraag: Wat is de 'simpele' manier om een één enkele regel tekst verticaal te centreren?

Antwoord: De simpele manier is door dit met de 'line-height' property te doen. Hiermee kan je de regel tekst handmatig op een hoogte zetten. Dit doe je in combinatie met 'vertical-align: center'.


Vraag: welke truc wordt vaak gebruikt om meerdere regels verticaal te centreren?

Antwoord: De CSS property 'display: table-cell' in combinatie met 'vertical-align: middle' wordt hiervoor vaak gebruikt.


Vraag: In 'styles.css' staan twee verschillende CSS regels die de height instellen van een element met class="area tall". Waarom krijgt de regel voor class 'tall' voorrang?

Antwoord: Ondanks dat beide classes even specifiek zijn wordt de regel voor class 'tall' toegepast omdat deze later in het CSS bestand staat.
